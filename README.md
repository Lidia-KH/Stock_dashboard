## Overview
StockPilot is a lightweight inventory management system designed for small and medium businesses.  
This version is a functional demo showcasing core stock management capabilities.

## Features
- Product management (Create, Update, Delete)
- Stock movements (IN / OUT)
- Low stock alerts
- Inventory value calculation
- Movement history tracking
- Dashboard KPIs


## Purpose
This demo illustrates the core architecture and business logic of the system.  
The production version can include:

- Multi-user authentication
- Role-based access control
- Multi-warehouse support
- Reporting & exports
- Cloud deployment
- API integrations
